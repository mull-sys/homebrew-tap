# homebrew-tap

Homebrew tap for [pufferhaus](https://github.com/pufferhaus) tools.

## Usage

```bash
brew tap pufferhaus/tap
```

Then install any tool:

```bash
brew install pufferhaus/tap/liste
```

## Tools

| Formula | Description |
|---------|-------------|
| [liste](https://github.com/pufferhaus/liste) | Portable CLI roadmap tracker for humans and AI agents |
